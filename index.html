What/Why

What

Woodcoin, or Logarithmic Bitcoin, is a decentralized, non-premined, mineable, proof-of-work global public currency. The unit of exchange is LOG, and it began in October 2014.

Whitepaper: English, 中文, Espagnol,Français, Deutsch

Why

Woodcoin was created to solve a problem which exists with the supply economics of many public coins. Public decentralized coins are Mineable. This is what makes them decentralized. Mining occurs with a reward schedule or a monetary policy. The reward schedule often drops too quickly (like Bitcoin and Litecoin) or too slowly (like Dogecoin and Ethereum).

Woodcoin’s reward decrease is a harmonic series, which more closely models renewable resources rather than non-renewable. This means that while the mining reward is always bigger today than it will be tomorrow, there will still always be an incentive to secure the network tomorrow, even with low transaction fees.

The logarithmic money supply curve is designed for stability and longevity, with the longest reward schedule of any other cryptocurrency. Half the LOGs will be released in the first 200 years, instead of the first 4 years (Bitcoin). However, every block is worth less than the preceding one - so earlier adopters are always rewarded and the supply will never rise above 28.5 million LOG.

The woodcoin system leads to the least premined, or most fair distribution, meaning that users are on a more equal footing than any other coin.

Technical Details

Woodcoin is a public cryptocurrency like Bitcoin. There are three major innovations, with design considerations listed here:

  • Proof of Work algorithm: Double Skein
      • This is the first consensus network built solely with the skein hash function. You can read more about Skein here. We hope to encourage further use and testing of the Skein family with this coin.
  • Elliptic Curve for Digital Signatures: X9_62_prime256v1.
      • To sign transactions and create public and private addresses we use the elliptic curve digital signature algorithm ECDSA just like Bitcoin. However we use a different curve as a basis to do so. This curve is perhaps the most used in the financial industry. Many people claim it is more secure. We like to encourage cryptographic diversity.
  • Logarithmic Coin Release
      • Woodcoin is the first coin to use a logarithmic coin release. The woodcutting reward is continually adjusted according to the harmonic series. From /src/main.cpp: 
      int64 static GetBlockValue(int nHeight, int64 nFees) {
	int64 nSubsidy = 1000000 * COIN;
	if (nHeight<100) return 1*COIN;
	else return nSubsidy/nHeight + nFees;
}
      This leads to a total money supply S which increases logarithmically in block height n: 
      log = natural logarithm
gamma = euler-mascheroni constant
forest = 5187377.51763962
S = 1000000* ( log n + gamma ) – forest
